# Book Files Organization

## Structure

```
books/
├── i-am-becoming-series/
│   ├── book-01-the-waking/       "The Waking" (Published Jan 23, 2026)
│   ├── book-02/                  (Rename when titled)
│   ├── book-03/                  (Rename when titled)
│   ├── book-04/                  (Rename when titled)
│   ├── book-05/                  (Rename when titled)
│   ├── book-06/                  (Rename when titled)
│   ├── book-07/                  (Rename when titled)
│   ├── book-08/                  (Rename when titled)
│   ├── book-09/                  (Rename when titled)
│   ├── the-companion/
│   └── series-shared/            Materials spanning the whole series
│
├── the-book-of-the-soul/
├── the-gift-of-this-day-1/
└── the-gift-of-this-day-2/
```

## Inside Each Book Folder

```
manuscripts/
  01-early-drafts/     First drafts, brainstorms, rough outlines
  02-revised-drafts/   Edited and reworked versions
  03-final/            Final manuscript ready for publishing

images/
  cover-art/           Cover designs, mockups, artwork
  interior/            Interior illustrations, photos, graphics

notes/                 Research, ideas, feedback, editorial notes
```

## File Types

- **Primary format**: Word documents (.docx)
- **Also used**: PDF (.pdf)
- **Google Docs exports**: If a file exists as both .docx and a Google Docs export, keep the .docx version and skip the duplicate unless the Google Doc has newer edits

## File Naming Suggestions

Use dates or version numbers to keep track of order:

- `the-waking-draft-2024-03-15.docx`
- `the-waking-v2-revised.docx`
- `the-waking-final-print-ready.pdf`
- `cover-concept-a.jpg`
- `cover-final-approved.png`

## How to Place Files

1. Identify which book the file belongs to
2. Decide if it's a manuscript, image, or note
3. Determine the stage: early draft, revised, or final
4. Drop it in the right folder
5. If a file spans the whole I Am Becoming Series, use `series-shared/`

## Renaming Book Folders

When you have titles for Books 02-09, rename the folders:

```
book-02/  -->  book-02-the-title/
```

Keep the number prefix so they always sort in order.
